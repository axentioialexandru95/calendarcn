## Starter Surface

<CalendarExample id="basic" />

Use this version when you want the smallest controlled integration that still
shows the full interaction model: local state, optimistic updates, direct event
creation, and the standard multi-view toolbar.

- Best for your first install of `CalendarRoot`
- Good base for product scheduling or internal team calendars
- Keeps all views available so users can switch between month, week, day, and agenda
